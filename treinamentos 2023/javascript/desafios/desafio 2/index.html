Desafio 2

Fazer um código que pergunte ao usuário 2 números, e com esses dois números executam as seguintes operações:

  • 2 + 2 = 4
  • 2 - 2 = 0
  • 2 x 2 = 4
  • 2 / 2 = 1
  • 2 % 2 = 0
  • 2 ^ 2 = 4
  • 2 é maior ou igual à 2: true
  • 2 é menor ou igual à 2: true
  • 2 é maior do que 2: false
  • 2 é menor do que: false

Resposta:

        
            var n1 = prompt("Qual é o primeiro número?")
            var n2 = prompt("Qual é o segundo número?")

            n1 = parseFloat(n1)
            n2 = parseFloat(n2)

            console.log(n1 + " + " + n2 + " = " + (n1 + n2))
            console.log(n1 + " - " + n2 + " = " + (n1 - n2))
            console.log(n1 + " x " + n2 + " = " + (n1 * n2))
            console.log(n1 + " : " + n2 + " = " + (n1 / n2))
            console.log(n1 + " % " + n2 + " = " + (n1 % n2))
            console.log(n1 + " ^ " + n2 + " = " + (n1 ** n2))

            console.log(n1 + " é maior ou igual à " + n2 + ": " + (n1 >= n2))
            console.log(n1 + " é menor ou igual à " + n2 + ": " + (n1 <= n2))
            console.log(n1 + " é maior do que " + n2 + ": " + (n1 > n2))
            console.log(n1 + " é menor do que " + n2 + ": " + (n1 < n2))
