Desafio proposto por mim

Escreva um código que imprima as n potências de 2 e seu somatório.

Resposta:

        
            var n = prompt("Digite n")

            n = parseInt(n)

            var soma = 0

            for (let i = 1; i <= n; i ++) {
                console.log("2 ^ " + i + " = " + (2 ** i))
                soma = soma + (2 ** i)
            }

            alert("Soma = " + soma)
