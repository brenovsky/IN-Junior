Desafio 1

Fazer um código que pergunte o nome de 2 usuários e suas idades,

depois mostre no console uma mensagem dizendo o nome e a idade do usuario.

Resposta: 

        
            var nome1 = prompt("Qual é o nome do usuário 1?")
            var idade1 = prompt("Olá, " + nome1 + "!" + " Qual é a sua idade?")
            var nome2 = prompt("Qual é o nome do usuário 2?")
            var idade2 = prompt("Olá, " + nome2 + "!" + " Qual é a sua idade?")
        
            console.log(nome1 + " tem " + idade1 + " anos.")
            console.log(nome2 + " tem " + idade2 + " anos.")
